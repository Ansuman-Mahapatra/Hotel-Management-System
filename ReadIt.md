# Hotel Management System  

A complete **Desktop-based Hotel Management System** built using **Pure Java Swing + MySQL** – fully functional with role-based access and real-time operations.

### Project Preview
![Dashboard](icon/dashboard.jpg)  

### Features
- Beautiful splash screen on startup  
- Secure **Login System** (Admin & Reception)  
- Manage Rooms: Add, Update, Search, Check Availability  
- Customer Management: Check-In, Check-Out, New Customer Registration  
- Employee & Driver Management  
- Department-wise operations  
- Live data display using JTable + rs2xml  
- Clean, modern, and colorful GUI with custom icons

### Technologies Used
- **Java SE** – Core language  
- **Swing + AWT** – Full GUI (JFrame, JPanel, JTable)  
- **MySQL** – Database storage  
- **JDBC** – Database connectivity  
- **rs2xml.jar** – ResultSet → JTable  
- **JCalendar** – Date picker (for check-in/out)  

### Entry Point
**`Splash.java`** → Starts the application with splash screen → Login → Dashboard
